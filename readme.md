# c++shell

A simple Unix shell written in C++. Supports basic commands, builtins like `cd`, `pwd`, and `echo`, and forks child processes to run all other commands.

## Usage
make
./src/main

## Commands
- `cd` - change directory
- `pwd` - print working directory
- `echo` - print text
- `exit` - quit the shell
- `ls` - list directory contents
- `cat` - print file contents
- `mkdir` - create a directory

# CPP_SHELL

## Features
### About
- A simple Unix shell written in C++.
### Commands
- `cd` - change directory
- `pwd` - print working directory
- `echo` - print text
- `exit` - quit the shell
- `ls` - list directory contents
- `cat` - print file contents
- `mkdir` - create a directory
- `<command>`   - runs any system command (ls, grep, cat, etc.)
### Extras
- Adding support for piping + better error output
- Added `help` to display list of commands
- Remove leading + trailing whitespace to better read commands
